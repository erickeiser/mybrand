Color Picker Game

View Project View Code

The goal of the Color Picker Game is to look at the RBG color provided and determine what the color is by clicking on the correct cirlcle. If you select the wrong circle it will disappear. But when you select the correct one, all of the colors will appear the same color. This game comes with both an easy and hard mode to help you perfect the RGB color range.
